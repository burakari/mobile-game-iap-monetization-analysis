# Mobile Game IAP Monetization Analysis

This project analyzes transaction-level in-app purchase (IAP) data from a mobile game.

## What this notebook includes

- Data loading and cleaning
- Core monetization KPIs (Revenue, Transactions, ARPPU)
- Platform comparison (iOS vs Android)
- Revenue distribution analysis

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib

## How to Run

Open the notebook:

mobile_game_monetization_engine.ipynb

Make sure required libraries are installed:

pip install -r requirements.txt
